# GURPS Roll Reactions

Модуль для Foundry VTT (система GURPS), который отображает визуальные реакции — GIF или картинки — в сообщениях чата в зависимости от результата броска.

## Возможности

- **Настройка на каждого персонажа** — у каждого актора свой набор реакций, настраивается прямо из листа персонажа
- **Три типа триггеров:**
  - **Навыки** — срабатывают, когда название навыка встречается в сообщении броска
  - **Оружие** — срабатывают по названию оружия, с отдельными реакциями для атаки, парирования и блока
  - **Универсальные** — срабатывают по любому произвольному тексту (атрибуты, вторичные характеристики, преимущества и т.п.)
- **Реакции по результату** — разные картинки для успеха, провала, критического успеха и критического провала
- **Каскадная логика** — если картинка для конкретного результата не задана, используется картинка по умолчанию
- **Тест реакции** — предпросмотр того, какая картинка появится при заданном уровне навыка и броске, без реального броска
- **Управление реакциями** — инструмент для GM: просмотр и удаление реакций у конкретного персонажа или сброс всех сразу
- **Локализация** — поддерживаются русский и английский языки

## Настройки

| Настройка | Область | Описание |
|---|---|---|
| Свечение при критических бросках | Для каждого игрока | Включает зелёную/красную рамку вокруг картинок при критическом успехе/провале |
| Управление реакциями | Только GM | Просмотр и удаление настроенных реакций по персонажам или сброс всех сразу |

## Совместимость

- Foundry VTT: v11–v13
- Система: GURPS (gurps)
